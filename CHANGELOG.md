# Changelog

The complete, per-release changelog — every version with its linked issues and
pull requests — is published on the
**[GitHub Releases page](https://github.com/Stashpeak/SimLauncher/releases)**.

That page is the single source of truth for what changed in each version, with
full notes per release. SimLauncher has shipped continuously since **v0.1.0
(2026-04-17)**. This file is intentionally a pointer rather than a second
hand-maintained changelog, so the two can't drift apart.

SimLauncher follows [Semantic Versioning](https://semver.org/spec/v2.0.0.html).
