## 2024-11-28 - Optimizing GameList.tsx renders\n**Learning:** The React render loop inside `GameList.tsx` was recalculating `hasActiveTitle` for every mapped game due to `Object.values(runningStatus).some(Boolean)` being inside the `.map` function, causing O(N^2) complexity.\n**Action:** Pulled the calculation of `hasActiveTitle` out of the loop since it evaluates the same value for all games.
